Install Wireguard 

  1. SSH into the droplet by opening a terminal and typing ssh root@[ip]

a.                  Where you would replace [ip] with the IP of your DigitalOcean Droplet

  1. Install the dependencies/packages needed to run docker. Commands: sudo apt install docker, sudo apt install docker-compose, sudo apt install wireguard
  2. Set up Wireguard using Docker Compose: 

a.                  Create a directory for Wireguard: 

mkdir -p /opt/wireguard
cd /opt/wireguard
 

  1. Create a docker-compose.yml file: 

nano docker-compose.yml
 

  1. Add the following configuration to docker-compose.yml: 

 

 

 

 

 

 

 

yaml  docker-compose.yml

  GNU nano 8.1                                                        

version: '3.8'

services:

  wireguard:

    container_name: wireguard

    image: linuxserver/wireguard

    environment:

      - PUID=1000

      - PGID=1000

      - TZ=American/New_York

      - SERVERURL=45.55.41.235

      - SERVERPORT=51820

      - PEERS=pc1,pc2,phone1

      - PEERDNS=auto

      - INTERNAL_SUBNET=10.0.0.0

    ports:

      - 51820:51820/udp

    volumes:

      - type: bind

        source: ./config/

        target: /config/

      - type: bind

        source: /lib/modules

        target: /lib/modules

    restart: always

    cap_add:

      - NET_ADMIN

      - SYS_MODULE

    sysctls:

      - net.ipv4.conf.all.src_valid_mark=1

 

  1. Save and exit. 



2.                  Run Wireguard: 

docker-compose up -d
 

3.                  Check logs to get the QR code: 

docker logs wireguard 

Test Your VPN

2.                  Before connecting: 

a.                  Visit IPLeak.net and screenshot your local IP. 

3.                  After connecting: 

a.                  Turn on the Wireguard VPN and revisit IPLeak.net.
